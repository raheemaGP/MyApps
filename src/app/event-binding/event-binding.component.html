Event Binding

Event binding in Angular allows you to listen for and respond to user events such as clicks, input changes, and form submissions. It is a one-way binding from the view to the component.

Syntax: (event)="handler"

In this syntax, event is the name of the DOM event you want to listen for (e.g., click, input, submit), and handler is the name of a method in your component that will be called when the event occurs.

Event binding is used to trigger component logic in response to user interactions. For example, you can bind to the click event of a button to execute a method that updates the component state or performs an action.

Examples

<button (click)="onButtonClick()">Click Me</button>

In this example, the click event of the button is bound to the onButtonClick() method in the component. When the button is clicked, the method will be executed.

Property Binding

Property binding in Angular allows you to bind a component property to an element property. It is a one-way binding from the component to the view.

Syntax: [property]="expression"

In this syntax, property is the name of the DOM property you want to bind to, and expression is a JavaScript expression that evaluates to the value you want to assign to that property.

Property binding is used to set the value of an element property, which can affect the behavior or appearance of the element. For example, you can bind to the disabled property of a button to enable or disable it based on a condition in your component.

Property Binding

Examples

<img [src]="imageSource" [alt]="imageAlt">

In this example, the src and alt properties of the <img> element are bound to properties in the component.
